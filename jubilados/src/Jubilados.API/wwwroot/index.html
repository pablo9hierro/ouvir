Carta de Correcao Eletronica (CCe)

      
Corrige informacoes de uma NF-e autorizada via evento tpEvento=110110. Valido em homologacao e producao.

      
A CCe NAO pode corrigir: valor total, CFOP, destinatario, data de emissao/saida. Use apenas para correcoes de texto em campos nao-criticos.

SPED EFD ICMS/IPI

      
Gera o arquivo SPED EFD (Registros 0, C e 9) com as NF-e autorizadas no periodo. Prazo mensal: ate o dia 15 do mes seguinte.

      
O arquivo gerado cobre as NF-e com cStat=100 (Autorizado) no periodo informado. Formato: texto delimitado por pipe (|).
